相关属性

a.字体：font-family

如：font-family: “微软雅黑”;

b.字体大小：font-size

如：font-size: 20px;
标题比较特殊，需要单独修改字体大小

c.字体粗细：font-weight

如：font-weight：700（加粗） 或 400（正常）

d复合属性

font: font-style（字体是否倾斜） font-weight（字体粗细） font-size/line-height（字体大小/行高） font-family（字体）

位置不可以改变
如： font: initial 700 16px/20px "微软雅黑"；

只有font-size和font-family不能省，其他都可以省

如：font:16px "微软雅黑"

文本属性

a.文本颜色：color

如：color:#ffoooo;或者color：red；

b.文本对齐：text-align(水平方向)

如：text-align: center;

c.装饰文本：text-decoration(下划线之类的)

如：text-decoration：underline

d.文本缩进：text-indent（首行）

如：text-indent：2em;或者text-indent：20px;

e.行间距：line-height

如：line-height：20px;
